# coherency

## Environment
We use the [Mamba](https://github.com/mamba-org/mamba) package manager.


```sh
mamba env create -f environment.yml
conda activate coherency
```

## Example script: 
An example script for running experiments can be found in [./run.sh](./run.sh)
